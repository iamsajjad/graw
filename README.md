<h1 align="center">
  Graw
</h1>

<p align="center">CLI tool to download only files or directories that you need from <b>github</b> or <b>gitlab</b></br>without need to clone the git repository</p>

## ‚ö°Ô∏è Quick start

First, [download](https://www.python.org/downloads/) and install **Python**. Version `3.7` or higher is required.

Installation is done by using the `pip install`

```bash
pip install graw
```
### How to Use graw ?

- `graw https://LINK-FOR-GITHUB-OR-GITLAB-FILE` ‚Äî to download one file from github repository
- `graw https://LINK-FOR-GITHUB-OR-GITLAB-DIRECTORY` ‚Äî to download one directory from github repository
- `graw -f https://LINK-FOR-GITHUB-OR-GITLAB` ‚Äî to download and overwrite exists content
- `graw -r https://LINK-FOR-GITHUB-OR-GITLAB-DIRECTORY` ‚Äî to download directory recursive [ all nested files ]
- `graw https://LINK-FOR-GITHUB-OR-GITLAB-DIRECTORY -t TYPE1 TYPE2 ... ` ‚Äî to download only files match extension types like `-t py`
